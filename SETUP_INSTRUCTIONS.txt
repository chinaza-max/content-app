MVC TypeScript App - Setup Instructions
========================================

1. Install Dependencies:
   npm install

2. Setup MySQL Database:
   mysql -u root -p
   CREATE DATABASE mvc_app;
   exit;

3. Configure .env file with your credentials

4. Build and Run:
   npm run build
   npm run dev

5. Access URLs:
   - Admin Login: http://localhost:3000/api/admin/login
   - Client Signup: http://localhost:3000/api/client/signup
   - Client Login: http://localhost:3000/api/client/login

Default Admin:
   Email: admin@yourapp.com
   Password: Admin@123

Next Steps:
- Copy all TypeScript code from the artifacts into src/ folders
- Copy all EJS templates into src/views/ folder
- Update .env with your database and email credentials
